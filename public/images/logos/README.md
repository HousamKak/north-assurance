# Logos

Place North Assurance logos here.

## Required Files

- `logo.svg` - Main logo (color version)
- `logo-white.svg` - White version for dark backgrounds
- `logo-dark.svg` - Dark version for light backgrounds
- `favicon.ico` - Browser favicon (16x16, 32x32, 48x48)
- `logo-192.png` - PWA icon 192x192
- `logo-512.png` - PWA icon 512x512

## Logo Specifications

- **Format:** SVG (vector) preferred
- **Alternative:** PNG with transparent background
- **Color scheme:** Should match primary brand colors
  - Primary: #015FC9
  - Secondary: #0DD3F1

## Usage

- `logo.svg` - Used in navbar and footer
- `logo-white.svg` - Used on dark backgrounds
- Favicons - Browser tab and PWA

# Team Photos

Place team member photos here.

## Required Images

Team member photos should be named according to the pattern:
- `firstname-lastname.jpg`

Example:
- `john-doe.jpg`
- `jane-smith.jpg`

## Specifications

- **Dimensions:** 400x400px (square)
- **Format:** JPG or WebP
- **File size:** < 100KB
- **Style:** Professional headshots with neutral background

## Current Team Members

## Fallback

If photos are not available, the UI Avatars API is used as a fallback with initials.
